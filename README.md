# gulp-create-test-files



## Installation

The easiest way is to install `gulp-create-test-files`
by running

```bash
npm install gulp-create-test-files -g
```


## Examples

### Basic

```bash
epsito example.js example.spec.js

epsito init

episto review

episto es6

episto angular2ts

episto report

episto forceawakens
 
 
});
```
